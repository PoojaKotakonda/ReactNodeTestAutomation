
# React + Node Test Automation Demo

## Prerequisites
- Node.js & npm
- Java JDK
- Maven

## Setup Instructions

### Backend
```bash
cd backend
npm install
node index.js
```

### Frontend
```bash
cd frontend
npm install
npm start
```

### Run Tests
```bash
cd tests/api
mvn test

cd ../ui
mvn test
```
